## Setup
### 1. Install dependencies

```sh
# Install expo
npm install -g expo-cli

# Clone the repo
git clone https://github.com/Kexon/weknowit-task
cd weknowit-task
```

### 2. Run

```sh
# Whilst inside project folder
npm start
```
